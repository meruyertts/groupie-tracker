### ABOUT:
Groupie Trackers consists on receiving a given API and manipulate the data contained in it, in order to create a site, displaying the information.

It will be given an API, that consists in four parts:

The first one, artists, containing information about some bands and artists like their name(s), image, in which year they began their activity, the date of their first album and the members.

The second one, locations, consists in their last and/or upcoming concert locations.

The third one, dates, consists in their last and/or upcoming concert dates.

And the last one, relation, does the link between all the other parts, artists, dates and locations.

Given all this a user friendly website is built where it displays the bands info through several data visualizations (examples : blocks, cards, tables, list, pages, graphics, etc).



### USAGE:
type "go run cmd/main.go" in the terminal
click on the link provided
